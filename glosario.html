TIPOS DE ETIQUETAS

En HTML tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

Estructura de Etiquetas
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales como texto img entre otros.

LISTA DE ETIQUETAS DIRIGIDAS AL TEXTO

    • <title> </title> sirve para ponerle nombre a la pagina
    • <head> </head> sirve para guardar el title
    • <h1> </h1> se utiliza para colocar un titulo
    • <p> </p> se utiliza para colocar un parrafo
    • <br> </br> se utiliza para saltar un parrafo
    • <label> </label> define el rotulo para un elemento
    • <ol> </ol> define una lista ordenada
    • <li> </li> define un articulo de una lista numerada
    • <form> </form> define un formulario
    • <body> </body> define el cuerpo del documento
    • <input> </input> define un control de entrada de texto
    • <article> </article> Representa una composición auto-contenida en un documento, página, una aplicación o en el sitio
    • <div> </div> Esta etiqueta permite agrupar varios elementos de bloque
    • <link> Ayuda a conectar el documento HT con el documento CSS.

  • color: cambia el color del texto

  • border-style: se utiliza para seleccionar el stilo delñ borde

listado de propiedades para caja
  • height: define la altura de la caja
  • width: define el ancho de la caja
  • border: permite establecer simultáneamente los cuatro bordes
  • margin: establece el margen para los cuatro lados
  • padding: establece el espacio de relleno requerido por todos los lados de un elemento
  • line-height: establoece la altura de una casilla remarcada por lineas
  • flex-direction: se utiliza para colocar el tipo de caja.En una caja flex podemos colocar los elementos en cualquier dirección
  • justify-content: Se utiliza para alinear los ítems
  • box-sizing: es una propiedad CSS para cambiar el modelo de caja por defecto de los navegadores
  • object-position: la propiedad object-position determina el alineamiento del elemento dentro de la caja
  • border-radious: permite definir que tan redondeadas seran las esquinas
